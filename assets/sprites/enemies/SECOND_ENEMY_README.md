# Second Enemy Sprites

This directory should contain the sprite sheets for the SECOND_BASIC enemy type.

Required sprite files:
- second_idle.png (4 frames, idle animation)
- second_walk.png (4 frames, walking animation)
- second_attack.png (4 frames, attack animation)
- second_hurt.png (2 frames, hurt animation)

Each sprite sheet should be a horizontal strip of frames.
Frame size: 64x64 pixels (same as basic enemy)
Format: PNG with transparent background

The game will automatically load these sprites when SECOND_BASIC enemies spawn.
